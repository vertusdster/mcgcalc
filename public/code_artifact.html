核心战略：国内外市场“物理隔离”

为了在享受开源生态便利的同时规避潜在的法律风险，我们设计了“软硬解耦”的商业架构。这一部分将解释为何国内销售“全家桶”而海外仅销售“硬件载体”是实现利润最大化与合规性的最佳平衡点。

国内：软硬一体 SaaS 模式

  • • 硬件：ESP32 + 墨水屏 + 3D/实木外壳
  • • 软件：汉化版 TRMNL 固件 + 本地化 API
  • • 盈利点：硬件差价 + 后续增值数据服务
  • • 合规：公开修改后的固件源码 (GPLv3)，闭源后端

海外：纯硬件/极客 DIY 套件

  • • 硬件：仅作为“可编程显示终端”发售
  • • 软件：出厂仅含基础驱动，不预装 UI 固件
  • • 盈利点：极高溢价的硬件差价 + 外壳材质溢价
  • • 合规：规避 TRMNL 商标侵权及外观设计专利

多维盈利空间与利润率模型

通过对比不同屏幕尺寸和外壳材质，我们可以清晰地看到：海外市场的溢价能力远超国内，而国内小尺寸产品仅能作为引流工具。

避免同质化竞争的关键在于赋予硬件特定的“身份”。以下是三个经过调研验证、具备极高溢价空间的细分应用场景。
